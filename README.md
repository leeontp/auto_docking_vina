# Molecular Docking with AutoDock Vina in Google Colab

This repository contains a **Google Colab notebook** for performing **molecular docking simulations** using **AutoDock Vina** in a simple and reproducible workflow.

## Overview

The notebook automates the main steps required for a protein–ligand docking study:

1. **Upload molecular structures**  
   Load receptor and ligand files.

2. **Structure preparation**  
   Prepare molecular files for docking, including format conversion and preprocessing.

3. **Docking configuration**  
   Define the docking search space (*grid box*) and simulation parameters.

4. **Run AutoDock Vina**  
   Execute the docking calculation to predict ligand binding poses.

5. **Results analysis**  
   Visualize docking scores and generated conformations for protein–ligand interaction studies.

## Features

- Easy-to-use workflow in **Google Colab**
- Automated setup for **AutoDock Vina**
- Protein–ligand docking execution
- Basic docking result analysis
- Reproducible computational pipeline

## Requirements

- A **Google account** to run Colab
- Receptor structure (e.g., `.pdb`)
- Ligand structure (e.g., `.mol2`, `.sdf`, `.pdb`)

## Usage

Open the notebook in **Google Colab**, upload your molecular structures, configure the docking parameters, and run the cells sequentially.

## Applications

This notebook can be used for:

- Protein–ligand interaction studies
- Drug discovery and virtual screening
- Computational chemistry education
- Molecular modeling research
